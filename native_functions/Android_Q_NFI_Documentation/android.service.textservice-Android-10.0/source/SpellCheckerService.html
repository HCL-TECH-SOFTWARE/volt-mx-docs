 </p>

 {@link android.view.textservice.SuggestionsInfo} for the input.
 You may want to override
 {@link android.service.textservice.SpellCheckerService.Session#onGetSuggestionsMultiple(TextInfo[], int, boolean)} for
 better performance and quality.
 </p>

 <p>Please note that {@link android.service.textservice.SpellCheckerService.Session#getLocale()} does not return a valid
 locale before {@link android.service.textservice.SpellCheckerService.Session#onCreate()} </p>
