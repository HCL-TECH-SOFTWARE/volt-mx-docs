/**@class java.lang.ref.Reference
@extends java.lang.Object

 implemented in close cooperation with the garbage collector, this class may
 not be subclassed directly.

 @author   Mark Reinhold
 @since    1.2
*/

*/

@return {Boolean} <code>true</code> if and only if this reference object has

*/

/**Ensures that the object referenced by the given reference remains

 this method.  Invocation of this method does not itself initiate garbage

 <p> This method establishes an ordering for

 that are implemented in ways that rely on ordering control for correctness.
@param {Object {Object}} ref the reference. If {@code null}, this method has no effect.
@param ref the reference. If {@code null}, this method has no effect.
@since 9
