      {@link android.net.wifi.aware.WifiAwareNetworkSpecifier.Builder}.
 </ul>

     {@link android.net.wifi.aware.DiscoverySessionCallback} indicate message was transmitted successfully,

@param {Number} messageId An arbitrary integer used by the caller to identify the message. The same
