
# Get the Image by ID

The Get Image by ID API is used to fetch the images by specific ID.

## URL

The HTTP URL for Get the image by ID API is:

```
http://<<host>>:<<port>>/service/noauthServices/noauth/assets/<id>
```

## Method

GET
